## 📸 HOW TO TAKE GOOD SCREENSHOTS

### For Homepage:
1. Open http://localhost:5173
2. Make sure page is fully loaded
3. Use full browser window (not just viewport)
4. Press F12 → Toggle device toolbar → Select "Responsive"
5. Set to 1920x1080
6. Take screenshot (Windows: Win+Shift+S)

### For Product Configurator:
1. Go to 4Over products page
2. Click "Customize & Order" on any product
3. Wait for modal to fully load
4. Take screenshot of the configurator

### For Mobile View:
1. Press F12 in browser
2. Click device toolbar icon
3. Select "iPhone 12 Pro" or similar
4. Take screenshot

### Tools for Screenshots:
- **Windows:** Snipping Tool (Win+Shift+S)
- **Chrome:** DevTools screenshot feature
- **Extensions:** Awesome Screenshot, Nimbus
- **Online:** Use browser's built-in screenshot

## 🎨 MAKING SCREENSHOTS LOOK PROFESSIONAL

### Before Taking Screenshot:
- [ ] Clear browser cache
- [ ] Use incognito mode (no extensions visible)
- [ ] Zoom to 100%
- [ ] Hide bookmarks bar
- [ ] Use clean browser (Chrome/Edge)
- [ ] Make sure all images are loaded
- [ ] Check for any errors in console

### After Taking Screenshot:
- [ ] Crop unnecessary parts
- [ ] Resize to reasonable dimensions
- [ ] Compress if file size is large
- [ ] Add subtle border (optional)
- [ ] Save as PNG or JPG

### Tools for Editing:
- **Windows:** Paint, Paint 3D
- **Online:** Canva, Photopea
- **Software:** GIMP (free), Photoshop
